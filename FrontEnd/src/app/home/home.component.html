Mentor-on-Demand Portal



   




About us

This is a software system which lets users search for a Mentor for a specific Technical Course, User can search, select, propose& finalize a Mentor of his Choice, post which Training gets started. Along with B2C, B2B functionality also need to be supported, for which Users can be Corporate Organizations seeking Technical Courses, which are offered by various Vendor Organizations(which may have pool of Trainers). Payment processing, E-mail interaction is within scope of the Project.Mentor need to SignUp to provide Mentoring service through the Portal
